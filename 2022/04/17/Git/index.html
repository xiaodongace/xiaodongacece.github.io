Git常用操作命令 - 代码提交和同步代码

  • 第零步: 工作区与仓库保持一致
  • 第一步: 文件增删改，变为已修改状态
  • 第二步: git add ，变为已暂存状态

  • 第三步: git commit，变为已提交状态

  • 第四步: git push，变为已推送状态
